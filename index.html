Vocabulary Flashcards

Variable Naming Conventions

Lowercase letter for first word and then a capital letter for the first of each concecutive word without spaces. likeThisVariable

Return

The command used to return a result from a function you write in JavaScript.

Strings

A variable that represents a string of text. It could be anything from one letter to a whole book! (size capped at 2^53 - 1 characters)

Booleans

A variable type that is either True or False

NaN

a numeric representation of the thing you entered not being a number.

Alert

A command you can run in JavaScript to cause a pop-up on the user's computer.
